# Kiro 専門エージェント完全ガイド

## はじめに

Kiro CLIには、開発プロセスの各フェーズを専門的にサポートする**28種類の専門エージェント**が用意されています。これらのエージェントは、単なるコード生成ツールではなく、**あなたの開発チームの一員**として、設計・実装・テスト・運用の全工程で高品質な成果物を生み出すパートナーです。

### サブエージェントとは？

サブエージェントは、特定の専門分野に特化したAIアシスタントです。メインのKiroに依頼すると、適切な専門エージェントが自動的に選ばれ、並列実行で効率的にタスクを処理します。

**主な特徴**:
- 🎯 **専門性**: 各分野のベストプラクティスを熟知
- ⚡ **並列実行**: 最大4つのエージェントが同時作業
- 🔒 **独立コンテキスト**: メイン会話を汚さず効率的
- 🤝 **連携**: 複数エージェントが協力して品質向上

## 🚀 使い方の例

### 例1: 新規プロジェクトの立ち上げ

```bash
# 1. 要件定義
kiro-cli chat --agent requirements-analyst "新規ECサイトの要件を整理したい"

# 2. システム設計
kiro-cli chat --agent architect "要件定義書を元にシステム設計をお願いします"

# 3. UI/UX設計
kiro-cli chat --agent uiux-designer "ユーザー体験を重視したUI設計をお願いします"

# 4. プロジェクト計画
kiro-cli chat --agent project-manager "WBSと工数見積をお願いします"
```

### 例2: 品質向上のための連携

```bash
# 実装完了後、複数の専門家にレビュー依頼
kiro-cli chat --agent security "セキュリティレビューをお願いします"
kiro-cli chat --agent performance "パフォーマンス最適化の提案をお願いします"
kiro-cli chat --agent qa "テスト戦略のレビューをお願いします"
```

### 例3: テスト自動化の完全フロー

```bash
# 1. テスト計画
kiro-cli chat --agent mobile-test-planner "iOSアプリのテスト計画を作成してください"

# 2. テストコード生成
kiro-cli chat --agent mobile-test-generator "テスト計画を元にXCUITestコードを生成してください"

# 3. テスト失敗時の修復
kiro-cli chat --agent mobile-test-healer "失敗したテストを修正してください"
```

## 💡 効果的な使い方のコツ

### 1. 適切なエージェントを選ぶ
- 各エージェントの専門分野を理解する
- タスクに最適なエージェントを選択する
- 複数のエージェントを組み合わせる

### 2. 並列実行を活用する
- 独立したタスクは並列実行で効率化
- 最大4つのエージェントが同時作業可能
- 依存関係のあるタスクは順次実行

### 3. 段階的に進める
- 設計 → 実装 → テスト → レビューの順で進める
- 各段階で適切なエージェントを活用
- フィードバックを次の段階に反映

### 4. ドキュメント化を忘れずに
- 重要な決定事項は必ずドキュメント化
- Scribeエージェントで文書作成
- チーム内での知識共有

