# アルゴリズム可視化ページ 設計書

## 概要

アルゴリズムの計算量(O記法)を視覚的に学べるインタラクティブページ。
2つのアルゴリズムを横並びで同時に実行し、スライダーでステップを操作して
計算量の違いを目で確認できるUI。

## アルゴリズム一覧(20問)

### ソート(4組)

1. バブルソート O(n^2) vs クイックソート O(n log n)
2. 選択ソート O(n^2) vs マージソート O(n log n)
3. 挿入ソート O(n^2) vs ヒープソート O(n log n)
4. バブルソート O(n^2) vs 挿入ソート O(n^2)

### 探索(2組)

5. 線形探索 O(n) vs 二分探索 O(log n)
6. 線形探索 O(n) vs ハッシュ探索 O(1)

### データ構造(4組)

7. 配列先頭挿入 O(n) vs 連結リスト先頭挿入 O(1)
8. 配列末尾挿入 O(1) vs 配列先頭挿入 O(n)
9. ソート済み配列挿入 O(n) vs BST挿入 O(log n)
10. スタック push/pop O(1) 可視化

### 数値/文字列(4組)

11. 文字列探索: 素朴法 O(nm) vs KMP O(n+m)
12. フィボナッチ: 再帰 O(2^n) vs DP O(n)
13. 最大部分配列: 全探索 O(n^3) vs カダネ O(n)
14. べき乗: 素朴 O(n) vs 繰り返し二乗 O(log n)

### 応用(6組)

15. 行列乗算: 素朴 O(n^3) vs ストラッセン O(n^2.81)
16. 重複検出: 二重ループ O(n^2) vs ソート+走査 O(n log n)
17. 二数の和: 全探索 O(n^2) vs ハッシュマップ O(n)
18. 回文判定: 素朴 O(n^2) vs 両端ポインタ O(n)
19. 素数列挙: 試し割り O(n sqrt(n)) vs エラトステネスの篩 O(n log log n)
20. 最短経路: ダイクストラ O(V^2) vs ダイクストラ+ヒープ O((V+E)log V)
