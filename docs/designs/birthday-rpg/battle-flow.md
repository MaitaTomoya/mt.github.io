## フロー図

```mermaid
flowchart TD
    MENU["バトルメニュー\nたたかう / わざ / どうぐ / にげる"]

    %% たたかう
    MENU -->|たたかう| ATK["通常攻撃実行\nダメージ計算（クリティカル/ミス/通常）"]
    ATK --> ATK_MSG["攻撃結果表示\n（Aで次へ）"]
    ATK_MSG -->|敵HP0| WIN["敵を倒した!"]
    ATK_MSG -->|敵HP残| ENEMY_ATK["敵の反撃\nダメージ計算"]

    %% わざサブメニュー
    MENU -->|わざ| SKILL_MENU["わざ選択（2x2 + もどる）\nかえんぎり / いなずま\nきあいだめ / ベホマラー\n　　　　　　　もどる"]
    SKILL_MENU -->|もどる| MENU
    SKILL_MENU -->|技選択| SKILL_ATK["技実行\nダメージ計算 or 回復"]
    SKILL_ATK --> SKILL_MSG["技結果表示\n（Aで次へ）"]
    SKILL_MSG -->|敵HP0| WIN
    SKILL_MSG -->|敵HP残| ENEMY_ATK

    %% どうぐサブメニュー
    MENU -->|どうぐ| ITEM_MENU["どうぐ選択\n持っているアイテム一覧\n　　　　もどる"]
    ITEM_MENU -->|もどる| MENU
    ITEM_MENU -->|アイテム使用| ITEM_USE["アイテム使用\nHP回復など"]
    ITEM_USE --> ITEM_MSG["使用結果表示\n（Aで次へ）"]
    ITEM_MSG --> ENEMY_ATK

    %% にげる
    MENU -->|にげる| RUN["逃げきれなかった!\n（まいた3回目: 裸イベント）"]
    RUN --> RUN_MSG["結果表示\n（Aで次へ）"]
    RUN_MSG --> ENEMY_ATK

    %% 敵の反撃
    ENEMY_ATK --> ENEMY_MSG["敵攻撃結果表示\n（Aで次へ）"]
    ENEMY_MSG -->|HP0| GAMEOVER["GAME OVER\nタイトルに戻る"]
    ENEMY_MSG -->|HP残| MENU

    %% 勝利
    WIN --> RESULT["勝利処理\n（ボス撃破）"]
```
