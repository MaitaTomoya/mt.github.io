# バースデー魔法陣カメラ v2 - 設計書

## 概要

インカメラで顔を検出し「今日の主役」テキストを頭上にトラッキング表示。
同時に友達の写真が上から降ってきて、パンチ（モーション検出）で吹っ飛ばすゲーム。

## 機能

1. **顔トラッキング**: face-api.jsで顔検出、頭上に「今日の主役」表示
2. **パンチゲーム**: 友達の写真が降ってくる → パンチで吹っ飛ばす
3. **撮影・ダウンロード**: スクリーンショット保存

## 技術構成

- 顔検出: face-api.js（tiny_face_detector、約190KB）
- モーション検出: フレーム差分比較（Canvas）
- 物理演算: 簡易重力+吹っ飛びベクトル
- カメラ: getUserMedia（facingMode: 'user'）

## パンチ検出ロジック

1. 前フレームと現フレームのピクセル差分を計算
2. 差分が大きい領域 = 動きがある
3. 動きの大きさが閾値超え → パンチ判定
4. パンチ位置と写真位置の距離がヒット範囲内 → ヒット

## 写真の挙動

- 上からランダムな位置・速度で落下
- ヒット時: パンチ方向に吹っ飛ぶ + 回転
- 画面外に出たら消滅 + 新しい写真生成
- スコア表示
